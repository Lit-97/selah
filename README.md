# Selah

Selah is a career guidance and job tracking platform designed to help users reflect, plan, and move forward in their professional journey.

## Features (Planned)

- Job application tracker
- Career planning dashboard
- Resume analysis
- GitHub activity integration
- AI career insights

## Tech Stack

Frontend:
- Next.js
- React
- TypeScript
- TailwindCSS

Backend:
- Node.js
- Express

Database:
- PostgreSQL
